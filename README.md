# Mini Reproductor WAV

Pequeño reproductor de archivos .wav desarrollado en Python.

## Descargar

Ir a la sección Releases y descargar la última versión o con la url:
https://github.com/21programado/reproductor/releases/latest y hacer clic en mini_reproductor.exe

## Cómo usar

1. Ejecutar el archivo .exe
2. Cargar un archivo .wav
3. Reproducir

⚠ Windows puede mostrar advertencia de seguridad porque el ejecutable no está firmado digitalmente.
